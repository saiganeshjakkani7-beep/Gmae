# Chain Bloom

Chain Bloom is a lightweight browser puzzle game built for GitHub Pages.

## Why this game

Current casual-game trends favor quick mobile sessions, one-thumb controls, no-download browser play, and hybrid-casual loops with simple rules plus light progression. Chain Bloom uses that pattern: connect matching tiles, score bigger chains, beat the target, and move to the next level.

## Play

Open `index.html` in a browser, or publish the repository with GitHub Pages.
